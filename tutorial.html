How to transcribe the postcards

Transcription of these postcards is relatively easy!

All we want you to do, is to enter details of the correspondance, transcribe the text as best you can (bearing in mind the variable hand writing) and if at all possible translate the contents.

The fields we would like you to transcribe are:

  • Date of card (if present)
  • The message
  • A translation if available
  • The possible/probable/definite language
  • The address to which it was sent
  • Any comments you might have

Dont worry if you cannot get everything. Each card will be worked on by several people.
